# Meta Ad Copy Variants — A/B Test Pack (Schwartz Framework)

**Owner:** Copywriter
**Purpose:** Ready-to-test alternative copy for each existing ad. Drop into Meta Ads Manager.
**Testing rule:** One variable at a time. Test headline vs headline, body vs body, never both.
**Update cadence:** New variants every 14 days.
**Language:** English only. Board mandate.
