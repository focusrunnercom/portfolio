# SMS Outreach Sequences — AI Patient Acquisition (Med Spa)

**Owner:** Copywriter
**Channel:** SMS/Text message (98% open rate, 45% reply rate)
**Tone:** Direct, peer-to-peer. No link-stuffing. Short sentences.
**Best time:** 12 PM (lunch break) or 6 PM (post-work) local time
**Compliance:** Include "Reply STOP to opt out" in first message. TCPA compliant.
**Language:** English only. Board mandate.

## Quick-Reference Cards

### Warm Nurture (3-touch)

```
T1: Ask about audit → offer instant qualification
T2: 78% stat → offer Loom demo
T3: Final offer → "reply go" CTA
```

### Cold Outreach (3-touch)

```
T1: Referral intro + short pitch → ask for time
T2: Bullet benefits + case study → 15 leads guarantee
T3: Market share FOMO → calendar link + breakup
```

### Post-Demo (4-touch)

```
T1: Thank you + deliverables → qualification question
T2: Custom mockup offer → low friction
T3: Built mockup → show the work done → offer
T4: Final close → "reply yes" CTA
```

### No-Show Recovery (3-touch)

```
T1: "Something came up" → send Loom
T2: Follow-up → answer questions
T3: Direct close → "reply go"
```

## Rules

1. **One sequence at a time per lead.** Don't overlap sequences.
2. **48-hour minimum gap between touches.** Text is invasive. Pace matters.
3. **If they reply at any touch → human takes over immediately.** Sales agent jumps in.
4. **Always include opt-out in first message.** TCPA requires it.
5. **Track:** reply rate, click rate, demo booked rate per sequence.
6. **No images or media in first SMS.** Links are okay in follow-up touches.
