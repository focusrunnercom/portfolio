# Case Studies

This directory contains case studies of FocusRunner client engagements.

## Case Study Template

Each case study should follow this format:

```markdown
# [Client Name] — [One-line Result Summary]

**Industry**: [e.g., SaaS, E-commerce, Healthcare]
**Service**: [Voice Agents / Workflow Automation / Email Marketing]
**Duration**: [e.g., 3 months]
**Team Size**: [e.g., 2 AI agents + 1 human PM]

## Challenge
[2–3 sentences describing the client's problem before engagement]

## Solution
[3–5 sentences describing what we built and why]

## Results
- [Metric 1 with percentage or absolute number]
- [Metric 2 with percentage or absolute number]
- [Metric 3 with percentage or absolute number]

## Tech Stack
- [Key technology 1]
- [Key technology 2]
- [Key technology 3]

## Client Quote
> "[Quote from client about the engagement]"

---

*Confidentiality note: Client names and proprietary data shared only with permission.*
```
